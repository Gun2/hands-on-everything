# Board API APP
간단한 게시판 CRUD API를 제공하는 애플리케이션

## APIs
| Method | Path         | Desc      |
|--------|--------------|-----------|
| GET    | /boards      | 모든 게시글 조회 |
| GET    | /boards/{id} | 게시글 상세 조회 |
| POST   | /boards      | 게시글 생성    |
| PUT    | /boards/{id} | 게시글 수정    |
| DELETE | /boards/{id} | 게시글 삭제    |
