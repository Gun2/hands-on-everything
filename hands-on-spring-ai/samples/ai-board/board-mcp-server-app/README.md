# Board MCP Server APP
게시판 API(board-api-app)를 AI가 직접 사용할 수 있도록 연결하는 Spring AI MCP 서버 애플리케이션

## Tools
| Name            | Desc      |
| --------------- | --------- |
| getBoardList    | 모든 게시글 조회 |
| getBoardDetail  | 게시글 상세 조회 |
| createBoardPost | 게시글 생성    |
| updateBoardPost | 게시글 수정    |
| deleteBoardPost | 게시글 삭제    |
